Sitio Web Estático Dockerizado

Clase 2: Dockerfiles - Nginx + HTML

Bienvenido

Este es un sitio web estático servido por Nginx dentro de un contenedor Docker.

Este ejemplo demuestra cómo:

  • Copiar archivos estáticos a una imagen Docker
  • Configurar Nginx con un archivo personalizado
  • Exponer puertos personalizados
  • Construir imágenes ligeras con Alpine Linux

Tecnologías Utilizadas

Docker

Contenedorización de la aplicación

Nginx

Servidor web de alto rendimiento

Alpine Linux

Imagen base minimalista (5 MB)
