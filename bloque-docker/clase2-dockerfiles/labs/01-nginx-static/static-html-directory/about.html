Acerca de este Lab

Ejemplo de sitio estático con Nginx

Propósito del Laboratorio

Este laboratorio demuestra cómo crear una imagen Docker personalizada para servir contenido estático HTML utilizando Nginx.

Conceptos prácticos:

  • Uso de la instrucción FROM con imágenes Alpine
  • Copia de directorios completos con COPY
  • Configuración personalizada de Nginx
  • Exposición de puertos no estándar
  • Construcción y ejecución de imágenes personalizadas

Ventajas de Alpine

La imagen base nginx:alpine ofrece:

  • Tamaño mínimo: aproximadamente 24 MB vs 142 MB de nginx:latest
  • Mayor seguridad: menor superficie de ataque
  • Inicio rápido de contenedores
  • Menor uso de ancho de banda
